# 💪 FitBot — ваш персональный фитнес-помощник в Telegram

**FitBot** — это open-source Telegram-бот, который помогает отслеживать тренировки, питание и прогресс в достижении фитнес-целей.  
Никакой магии, только чистая математика и ваша дисциплина.

## 📋 Возможности

- 🧑‍🎓 **Регистрация профиля**: пол, возраст, рост, вес, уровень активности
- 🎯 **Расчет норм КБЖУ**: автоматический подсчет калорий, белков, жиров и углеводов
- 🏋️ **Дневник тренировок**: добавление упражнений, просмотр статистики
- 🍎 **Дневник питания**: внесение продуктов, отслеживание калорий
- 📊 **Прогресс**: графики изменения веса и объемов
- 📅 **Напоминания**: уведомления о тренировках и приемах пищи
